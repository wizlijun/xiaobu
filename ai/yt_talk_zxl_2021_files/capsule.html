视频号：连接的下个十年

视频化表达是未来十年内容领域的主体。视频号不是战略任务，而是为了解决“人人皆可创作”的挑战。

什么是视频？

未来的视频不应该只是一个文件（File），而应是云端化的结构化数据。它包含创作者信息、评论、互动，且能被分享和流转。

视频号的 ID 意义

点击查看解析

微信的价值在于ID（身份）。视频号不仅是内容，更是一个公开领域的身份（ID），是机构和个人在微信里的官方网站。

1:2:10 理论

点击查看预测与现实

预测：关注:朋友赞:机器推荐 = 1:2:10。
现实：目前朋友赞还是大头。
观点：机器推荐（舒适区/娱乐）未来占比会越来越大。

为什么起步阶段很难？ +

视频号早期是死循环：内容不好 -> 没浏览量 -> 没好内容。直到引入社交推荐（朋友点赞），才跨过了生死线。社交推荐是微信独特的优势。

全屏 vs 信息流 +

一屏显示几条内容取决于命中率。命中率低时（如关注流），需要一屏多条供选择；命中率高时（如推荐流），全屏沉浸体验更好。

直播：个人的终极表达

直播比短视频门槛更低，更轻松。短视频需要准备，直播就像聊天。

未来点开一个人的名片，可能直接看到他正在直播的画面（如眼镜视角），看到他所看到的世界。

春节与直播

点击查看场景

传统的拜年是面对面的。直播拜年比发文字、红包更接近线下拜年的体验。

有趣的实验性功能

产品是关于创意的，99%的想法可能不靠谱，但靠谱的那1%会很有趣。

  • 状态 (Status) +

    名片太死板了。“社交的本质是找到同类”。通过状态（如“发呆”、“数羊”），你可以看到谁和你处于同一状态，从而产生连接。

  • 听歌视觉化 +

    听歌时应该看到什么？不该是旋转的唱片。应该看到别人听这首歌时眼前的画面，实现想象力的连接。

  • 微信输入法 +

    缘起：用户投诉微信窃取聊天记录推广告（其实并没有）。
    目的：为了证明隐私安全性，也为了探索人机交互的第一个入口。

  • 浮窗 & 表情 +

    浮窗：新方案比旧的“狗皮膏药”优雅十倍。
    表情：炸弹屏幕裂开的效果，是为了模拟更强烈的情绪表达。

产品哲学

"连接" 与 "简单"

Connection & Simplicity

关于“简单” +

现在的微信和十年前看起来一样简单，虽然多了很多功能。“简单是终极的复杂”。简单是为了美感、实用和合理，而不是为了简单而简单。

关于“连接” +

微信做的是最底层的设施。重心不是做内容，而是做连接（人、内容、服务、货币）。连接意味着去中心化。

关于“速度” +

“其疾如风，其徐如林”。微信并不慢，小团队（如视频号团队仅一两百人）依然可以保持高效的创造力。
